Welcome to the Sentiment Analysis API

This project demonstrates a sentiment analysis application using the IMDb movie review dataset. The goal is to classify reviews as positive or negative while showcasing expertise in AI, ML, and MLOps . The process involves:.

  • Dataset preparation: Preprocessing IMDb reviews for feature extraction.
  • Model training: Using Logistic Regression, LSTM, CNN, or BERT.
  • Flask API: Building a REST API to deliver predictions.
  • Dockerization: Ensuring scalability and portability.
  • CI/CD pipeline: Automating builds, testing, and deployment.
  • Deployment: Hosting the API on a cloud platform like AWS or GCP.
